What is GitHub?

GitHub is a free online service that allows people to use git to create code repositories, which are containers for code. A link to GitHub.

Things covered in the next section:

  1. What is a branch?
  2. What is a commit?
  3. What is a pull request?

A branch is a section of code that has been seperated from the main branch of code, so that you can can work making changes without distrubing the main or master code. A branch should have a new and different name to the main code, every repository can one or more branches.
DO NOT MESS WITH THE MAIN BRANCH!

A commit is a save point for your code, while writing your code you can commit it to GitHub. The only helpful thing that happens once you commit multiple sections of code you make a breadcrumb trail or history of the previous code you've written, which is helpful for you and anyone else working on the project.

A pull request is a request made for the new section of code that you or someone in your group has written and would that code to be merged into the main or master code. A pull request will show up as a notification to others in the group, and then the changes proposed will be discussed, from here other people in the group can post feedback on the pull request and even add their own tweak with thier own commits.

There is a GitHub desktop app, that in my short personal experience I have found it to be very help, showing things that are gonig to added and thing you have removed. You can easily use this a starting point for you offline code and then commit and branches you have made. I highly reccommend

What is Unix Shell?

Unix shell is a great way to manipluate folder and files, wether that me making new ones or move already excisting one around. On windows it simple be accessed by opening a commmand window (windows key + cmd), then typing powershell. For Mac you can use a Bash is a good option.

Thing covered in the next section:

  1. How to show and change the working directory.
  2. How to list the files in a directory.
  3. How to create a new directory.
  4. How to create and remove files.

Once you have your shell open andready to navigate between directories, you can first by shwoing the current directory you are in by typing pwd(print working directory), if you would to then change the directroy you are the command "cd + directory" name to move into the directory, if you wanted to move back a directory you would use "cd .." which would take you back one.

You can list files in the current directory use the command "ls" you can the see which files and directories you like to manipulate.

Creating a new directroy is also pretty straight forward you would use the code "mkdir + whatever name you'dlike to call it" try and avoid spaces where youcan as it becomes difficult to manouver between them.

Creating a text file you would use the command nano nameoffile.txt and this is where you would write out what is inside your.txt file, once you are done save the file follwoing the shortcuts at the bottom of the terminal, the "^" symbol just mean either ctrl on windows or command on mac. To then remove that file you simply use the command "rm" ReMove plus the name of the file.

These are some of the very basic things you can do with shell, it can be a very powerful tool and learning how to manipluate a terminal with the need of a mouse can make for very quick and efficent work.
